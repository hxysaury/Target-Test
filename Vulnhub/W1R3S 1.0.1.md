### 主机发现

```bash
sudo nmap -sn 10.9.75.0/24 -oA
sudo arp-scan -l
```

### 端口扫描

简单扫描

```bash
sudo nmap -sT --min-rate 10000 -p- 10.9.75.11 -oA scan/ports

-sT TCP扫描
--min-rate 指定最低速率扫描，每秒发包速率，
-p-   指定1-65535 全端口扫描
-oA 指定扫描结果的存放位置,有三种不同的输出格式，会存储在scan文件下，命名为ports
```

![image-20231203160419583](./W1R3S%201.0.1.assets/image-20231203160419583.png)

查看端口扫描的输出结果

![image-20231203161007277](./W1R3S%201.0.1.assets/image-20231203161007277.png)

如果在扫描来端口过多的情况下，可以使用命令把扫出的端口单独提取出来，

```bash
grep open ports.nmap | awk -F'/' '{print $1}' | paste -sd ','


paste -s表示合并到一行，-d是指定分割符
```

效果展示：

![image-20231203161507698](./W1R3S%201.0.1.assets/image-20231203161507698.png)

也可以把上面的命令指定给一个变量

```bash
ports=$(grep open ports.nmap | awk -F'/' '{print $1}' | paste -sd ',')
```
